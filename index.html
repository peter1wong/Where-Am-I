    Where Am I app allows user to send their current location via SMS or eMail

    Project maintained by peter1wong Hosted on GitHub Pages — Theme by mattgraham

    Where Am I app allows user to send their current location via SMS or eMail. This is handy when you have no idea where you are. It will extract a physical address and its geolocation and form a SMS or eMail.
